# PharmaPOS Frontend

React-based frontend application for the PharmaPOS Pharmacy Billing System.

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Project Structure

- `src/components/` - Reusable UI components
- `src/pages/` - Page components (Login, Purchase, Sales, etc.)
- `src/services/` - API service layer
- `src/utils/` - Utility functions
- `src/constants/` - Configuration constants

## Features

- User authentication
- Company management
- Stock management
- Purchase and Sales transactions
- Invoice generation and reports
- Responsive design with Tailwind CSS
